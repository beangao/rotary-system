# Design Viewer

`docs/design/` フォルダ内の画面デザインをブラウザでプレビューするためのツールです。

## 起動方法

```bash
cd docs/design-viewer
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開くと、デザイン一覧が表示されます。

## 機能

- **一覧表示**: Club Admin / Mobile App のデザインをカテゴリ別に表示
- **個別プレビュー**: 各デザインをクリックしてフル画面でプレビュー
- **ホットリロード**: デザインファイルを編集すると自動的に反映

## 対応デザイン

### Club Admin
- ログイン・ダッシュボード
- イベント・例会管理
- 会員管理
- 管理設定
- お知らせ管理

### Mobile App
- アプリ起動画面
- ログイン画面
- 認証コード入力
- 新規登録フロー
- パスワードリセット
- プロフィール設定
- メインアプリ
- マイページ・プロフィール
- 会員名簿

## 新しいデザインを追加する場合

1. `docs/design/` にJSXファイルを追加
2. `src/App.jsx` の `designs` オブジェクトと `designList` 配列に追加
